Software Information

This application, M.A.C.R.O. (Mouse And Keyboard Rapid Operator), is designed to automate mouse and keyboard actions. It allows users to perform repetitive tasks with ease by simulating mouse clicks and keyboard presses.

For more information and to access the source code, visit the GitHub repository.

Submit issues/request here GitHub issues.

Python packages used:

  • PySide6: Used for creating the graphical user interface (GUI).
  • pynput: Used for controlling and monitoring input devices (mouse and keyboard).
  • keyboard: Used for detecting and responding to keyboard events.
  • json: Used for saving and loading application settings in JSON format.
  • os: Used for interacting with the operating system, such as file path manipulations.

Functions of the Software

  • Autoclick: Automatically clicks the mouse at a specified frequency.
  • Hold Click: Holds the mouse button down for a specified duration.
  • Keyboard Key Selection: Allows users to select a keyboard key to be pressed automatically.
  • Settings Management: Saves and loads user settings, such as selected key, frequency, and device used.
